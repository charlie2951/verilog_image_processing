# Basic Image operation in Verilog (Grayscale image only)
## Simulation only, not tested on FPGA
This code takes 64x64 grayscale image array and perform various image operation (invert, brightness and threshold operation). 
### File details
`convert_image.py`: used to create 32 bit hex value of user defined input image (see image file name inside script, rename your image as per that). 
A total 64x64 byte generated and they are stored into array of 4 byte each. <p>
`image_datagen.py`: A test image pattern generated by python code without any input image. <p>
`show_results.py`: used to plot original and processed image generated by Verilog testbench.<p>
`image_ip.v`: RTL code for different image related operation such as invert, brightness and threshold operation. You can change brightness amount in RTL code, default value set to 30.
